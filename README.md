# security-agent

> Static security analysis agent — OWASP Top 10, secrets scanning, CVE matching.

## What it does

Scans source code for injection vulnerabilities, hardcoded secrets, insecure dependencies, and known CVEs. Generates a prioritised remediation report with fix suggestions.

## Quickstart

```bash
pip install security-agent-ai
```

## Usage

```bash
security-agent scan src/ --report security-report.md
```
